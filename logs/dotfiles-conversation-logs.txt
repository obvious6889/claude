● For an air-gapped Red Hat machine you have a few options depending on what access you have:

  Option A — Transfer via USB/SCP from a machine that has internet

  On a machine with internet, download the binaries:
  # Download starship binary
  curl -sS https://starship.rs/install.sh | sh -s -- --yes --bin-dir ./

  # Download nvim appimage (self-contained, no install needed)
  curl -LO https://github.com/neovim/neovim/releases/latest/download/nvim-linux-x86_64.appimage
  chmod +x nvim-linux-x86_64.appimage

  Then copy everything to the air-gapped machine via USB or scp:
  scp starship nvim-linux-x86_64.appimage user@redhat-machine:~/.local/bin/

  On the Red Hat machine:
  mv ~/.local/bin/nvim-linux-x86_64.appimage ~/.local/bin/nvim

  Option B — Transfer dotfiles without git
  # On a machine with internet, bundle the dotfiles
  tar czf dotfiles.tar.gz -C ~ dotfiles/

  # Copy to Red Hat machine
  scp dotfiles.tar.gz user@redhat-machine:~/

  # On Red Hat machine
  tar xzf dotfiles.tar.gz
  cd ~/dotfiles && bash install.sh

  The install script will skip Homebrew (Linux) and just symlink the configs.
